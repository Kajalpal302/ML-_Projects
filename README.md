## SMS Spam Classifier

-An end-to-end Machine Learning and Natural Language Processing project that classifies SMS messages as Spam or Not Spam using TF-IDF Vectorization and a Multinomial Naive Bayes classifier. The project covers the complete ML lifecycle, including data cleaning, exploratory data analysis, NLP preprocessing, feature engineering, model training, evaluation, model serialization, and and deployment of a Streamlit web application on Render.
